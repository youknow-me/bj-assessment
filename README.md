# REST API

It processes various data types (Fibonacci, Prime, LCM, HCF) and integrates with Google Gemini AI for logic-based responses.

## Features
- **Data Processing**: Handles logic for Fibonacci sequences, Prime numbers, LCM, and HCF.
- **AI Integration**: Uses Google Gemini Pro/Flash to generate single-word answers for queries.
- **Production Ready**: Includes strict input validation, error handling, and standard status codes.
- **CORS Enabled**: Accessible from any client.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **AI**: Google Generative AI (Gemini)
- **Deployment**: Vercel
